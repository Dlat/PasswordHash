# PasswordHash
### Description
Hash any entered text.  
Intended for passwords only.

### Hashes:
- SHA512
- SHA384
- SHA256
- RIPEMD160
- SHA1
- MD5

### Built on:
- Net Framework 4.5.2
- Visual Studio 2015
- Tested on Windows 10 Build 14393

### Information Sources:
